## Bright-Line Rule

Before applying ANY edit to approved or finalized content, run three checks:
1. **Multi-paragraph scope:** Changes more than one paragraph? → PAUSE for operator approval.
2. **Analytical claim alteration:** Changes, removes, or reframes an analytical claim? → PAUSE for operator approval.
3. **Sourced statement modification:** Alters a statement attributed to a source? → PAUSE for operator approval.

If ANY check is true, the edit MUST NOT be applied without explicit operator approval. Log the decision to `/logs/decisions.md`.
