# Instruction B — Client-Facing (Inline Attribution)

## Attribution Rules — When to Cite

- REQUIRED for: specific numbers, counterintuitive findings, authority-building claims
- OPTIONAL for: general context, widely accepted dynamics, multi-source synthesis
- Never cite more than two sources inline for the same claim
- Format: organization name and year in running text (e.g., "according to Bain (2024)")

## Grade Filtering

- Only attribute High and Medium grade sources in prose
- Low grade: Sources Consulted only, never in prose attribution

## Conflict and Gap Handling

- Conflicted claims: note range, attribute to most authoritative, do not dramatize
- Known gaps: hedge language only ("early indicators suggest"), do not flag explicitly

## Supplementary Item Citation

- Attribute only if item contains a specific number or benchmark
- General framing context from supplementary items goes unattributed

## End Matter

- "Sources Consulted" section grouped by topic at end
- All sources including Low grade appear here

## Final

Strip all claim IDs from output prose.
